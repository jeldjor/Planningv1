# Planning-GJsystems v10.6

Basis: Planning-GJsystems v10.5-corrected.

## Installatie

1. Upload alle bestanden uit deze ZIP naar de root van de GitHub Pages-repository en vervang de bestaande bestanden.
2. Voer `SUPABASE_V10_6_COMPLETE.sql` één keer volledig uit in Supabase SQL Editor.
3. Log op laptop en iPhone volledig uit en daarna opnieuw in.
4. Test eerst het beheerdersaccount en daarna een gewoon gebruikersaccount.

`SUPABASE_V10_6_COMPLETE.sql` is herhaalbaar opgebouwd en is vóór oplevering tegen de echte Planningv1-databasestructuur uitgevoerd binnen een teruggedraaide testtransactie.

## Belangrijk

De v10.6 SQL is verplicht. Zonder deze migratie werken gebruikersgebonden daginstellingen, onderhoud per werkruimte en contactbijlagen niet volledig.
